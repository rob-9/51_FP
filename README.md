## MIPS Assembly Sudoku
### ICS51 Final Project -- Dr. Wongma

#### Setup
- `java -jar ICS51_Mars_F25_Sudoku.jar`
- Click **ICS51 Sudoku** under **Tools**
- Click **Connct to MIPS**

#### Sections
- Sudoku Board & Gameplay
- Loading a Game
- Performing a Player's Move
- Providing Hints
- Displaying Win Image
- Save Game Functionality

#### Goals
- Practice writing MIPS functions calls and nested function calls.
- Practice following MIPS register conventions.
- Experience working with the stack & memory organization & data types.
- Work with 2D arrays.
- Interacting with input/output files (system calls).
- Improve your understanding of MIPS aseembly instructions and writing MIPS assembly.
